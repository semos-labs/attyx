# Attyx v0.3.14

## Agent Tab Interaction & Status Updates

This release fixes agent tabs swallowing window UI clicks and adds live status indicators for supported agent sessions.

### What changed

- **Tab and split-pane clicks now keep working while agent apps use mouse tracking** — when OpenCode or other mouse-aware terminal apps enabled PTY mouse reporting, clicks meant to switch tabs or focus another split pane could be intercepted before Attyx handled them. Native tab and pane focus clicks now win first, so the window UI stays usable during agent sessions (#210)
- **Supported agent tabs now show live status dots** — OpenCode and Claude tabs now display status indicators directly in the tab strip, with detector-based matching that makes it easier to add more agents later. Waiting prompts, active work, and idle sessions are easier to distinguish at a glance (#210)
