# Attyx v0.3.8

## Xyron Integration

This release introduces first-class integration with [Xyron](https://github.com/semos-labs/xyron) — a modern shell built in Zig with Lua scripting, SQLite history, and structured data. Xyron replaces bash/zsh/fish with a purpose-built shell that's under 2MB.

When Xyron is installed and enabled, Attyx launches it in headless mode and communicates over a binary protocol instead of raw PTY I/O. This enables native UI features that aren't possible with a traditional shell:

- **Block-based output** — each command's output is wrapped in a bordered card with its input line, working directory, and exit status. Running, succeeded, failed, and interrupted states are visually distinct (#190)
- **Native completions** — Xyron pushes tab-completion candidates to Attyx over IPC, rendered as a cursor-anchored dropdown overlay instead of terminal text (#190)
- **Input buffering** — when idle, keystrokes are buffered in Attyx with cursor movement, then sent as a complete command on Enter. While a command is running, input flows directly to the process (#190)
- **Prompt rendering** — Xyron's prompt is fed through Attyx's terminal engine for full ANSI rendering, keeping prompt themes working as expected (#190)

### Configuration

Add to your `attyx.toml`:

```toml
[xyron]
enabled = true
# path = "/usr/local/bin/xyron"  # optional, searches PATH by default
```

Xyron is auto-detected from PATH or the configured path. macOS and Linux only for now.
